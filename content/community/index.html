The Apache Usergrid community has members in 170+ cities in 30 countries on 6 continents

Mailing-lists

Users

Perfect if you build apps against Apache Usergrid or deploy Apache Usergrid.

To subscribe to the user list, send a blank email to
user-subscribe@usergrid.incubator.apache.org.

Dev

If you are building contributions & modifications to Apache Usergrid’s stack, portal, cli and SDKs, this is the list for you.

To subscribe to the dev list, send a blank email to
dev-subscribe@usergrid.incubator.apache.org.

You can also read the archives.

Commits

This list receives an email whenever new code is committed to Apache Usergrid.

To subscribe to the commits list, send a blank email to
commits-subscribe@usergrid.incubator.apache.org.

You can also read the archives.
